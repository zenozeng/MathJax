# MathJax

MathJax's CDN is a little bit slow in China.
This repo is a self hosted MathJax plus configuration for color and XyJax.

## Usage

Include this in your markdown:

```html
<script src="//zenozeng.github.io/MathJax/init.js"></script>
```
